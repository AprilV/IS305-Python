# PowerShell Project Prototypes

This folder contains incremental prototypes showing project progress.

---

## Prototype 1 - Import, Filter, Display (February 15, 2026)

**Skills Demonstrated:**
- Lesson 1: Variables, Write-Host, Display
- Lesson 2: Import-Csv, Count
- Lesson 3: Where-Object, Filtering

**Functionality:**
- Imports 10,254 construction records
- Counts total projects
- Filters by Status (Open, Closed)
- Displays basic report

**To Run:**
```powershell
.\Prototype1_Import_Filter_Display.ps1
```

**Skills Demonstrated:**
- Lesson 4: Measure-Object, Group-Object, Statistics
